# project_se
this project for se_project

How to install

After clone this project

1.cd to soa_project

2.
```sh
composer install
```
3.create .env file by copy content .envexample

4.
```sh
npm install
```
5.
```sh
npm run dev
```
6.
```sh
npm run build
```
7.
```sh
php artisan key:generate
```
8.
```sh
php artisan serve
```
